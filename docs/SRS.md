# Software Requirements Specification (SRS) - MzansiBuilds

## 1. Introduction
MzansiBuilds is a developer platform designed for the Derivco Code Skills Challenge. It enables developers to build publicly, track project milestones, and collaborate.

## 2. Tech Stack Requirements
* **Frontend:** Next.js, Tailwind CSS
* **Backend:** Node.js, Next.js API Routes
* **Database:** PostgreSQL via Prisma ORM
* **Design Theme:** Green, White, and Black

## 3. User Stories (Functional Requirements)
* **Auth:** As a developer, I can create and manage my account.
* **Project Creation:** As a developer, I can create a new entry for a project, including its current stage and support required.
* **Live Feed:** As a developer, I can view a live feed of other projects and comment or raise a hand for collaboration.
* **Milestones:** As a developer, I can continuously update my project progress with milestones.
* **Celebration Wall:** As a developer, I am added to a Celebration Wall upon completing a project.
